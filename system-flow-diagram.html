🔄 Request Flow

User query → React frontend → FastAPI (JWT auth) → AI Service orchestrates: intent classification via Groq, data fetch via Query Engine, response generation via Groq → formatted response with optional chart back to frontend.

📊 Daily Snapshots

Snapshot Service runs on server startup, backfills last 30 days of aggregated metrics (revenue, orders, top items, peak hours) into daily_summaries table. Query Engine pulls this data for comparison queries.

🧠 Dual LLM Usage

Groq is called twice per request: (1) Intent classification with few-shot examples → JSON output, (2) Intelligent response generation with full schema context, retrieved data, and conversation history.
