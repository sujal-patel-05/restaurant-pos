Production Architecture

SujalPOS AI Architecture

Intelligent Restaurant POS System — A multi-layered architecture combining React, FastAPI, Groq LLM, and SQLite for real-time operations and AI-driven insights.

01 — Overview
Request Lifecycle
01
🧑‍💻

User Input

User types a natural language query in the Ask AI chatbot interface (React frontend).

02
🧠

Intent Classification

Groq LLM classifies intent (sales_query, inventory_query, etc.) with entities and confidence score.

03
🗄️

Data Retrieval

Query Engine fetches structured data from SQLite database using classified intent and entities.

04
💬

Smart Response

Groq LLM generates a context-rich, formatted response with tables, charts, and comparisons.

03 — Data Flow
Ask AI Chat Pipeline
User Message → Intelligent Response
1
Frontend sends message with conversation_id via POST /api/ai/chat
AskAI.jsx → api.js
2
JWT Authentication validates user identity and extracts restaurant_id
routes/ai.py
3
Intent Classification hybrid: rule-based first → Groq LLM fallback with few-shot examples
ai_service.py → groq_service.py
4
Query Engine maps intent → SQLAlchemy queries + daily_summaries for comparisons
query_engine.py
5
Database Query fetches orders, inventory, menu, wastage, daily_summaries
SQLite + SQLAlchemy
6
LLM Response Generation Groq processes data with system prompt, schema context, and persona
groq_service.py
7
Chart Parsing extracts __chart__{...}__chart__ blocks → structured chart_data JSON
ai_service.py
8
Frontend Rendering markdown + tables + Recharts visualization + copy button
ChatMessage.jsx
04 — Analytics Engine
Daily Snapshot Pipeline
A
⏰

Startup Trigger

On every server start, backfills missing days (last 30 days) for all restaurants automatically.

B
📊

Aggregation

Computes revenue, orders by type, payment modes, wastage, top item, peak hour per day.

C
💾

Upsert Storage

Stores in daily_summaries table (idempotent). Today refreshed every startup for accuracy.

D
🆚

Comparison Queries

LLM uses stored snapshots for: today vs yesterday, weekly trends, monthly reports.
